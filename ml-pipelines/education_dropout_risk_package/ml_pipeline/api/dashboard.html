How These Values Are Calculated

Education Records = count of rows for this resident in education_records.csv up to the cutoff date used for training.

Attendance Change is shown as: baseline attendance to pre last average attendance.

Baseline Attendance: the resident’s first recorded attendance_rate.

Pre Last Average Attendance: the average of all attendance rates before the final record. If only one record exists, it falls back to that one record.

Group Median: the median value for that feature across all residents included in the model training dataset.

The dropout risk score is generated by a trained random forest model. The values shown here are supporting context, not the full calculation.
